== Description ==

Wall Clock give you total five types clock to show in your website. Administrator can customize it from dashboard easily also it is user friendly and easy to integrate.
It is totally free to download and use.

== Installation ==

1. Downlaod wall-clocks.zip
1. Upload wall-clocks.zip to the plugin from wordpress admin
2. Activate the plugin
3. Visit Wall Clock from dashboard and configure
4. Use this short code [asclock] to show the clock
